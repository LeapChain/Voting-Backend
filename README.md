# Voting-Backend

## Getting started

### Docker

Copy the contents of `.env.example` file to `.env` and update the environment variables.

Build and run the image

```shell
docker-compose up -d --build
```

Now navigate to `http://127.0.0.1:3000/` and the server should be running.
